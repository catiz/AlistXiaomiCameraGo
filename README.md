### AlistXiaomiCameraGo
通过Alist/Openlist上传小米监控SMB存储的某日视频到任意云盘

使用方法：在config.yaml填入Alist/Openlist的信息

|       openlist       | openlist地址 |
|:--------------------:|:----------:|
|       username       |    用户名     |
|       password       |     密码     |
|xiaomiCameraVideosPath| 小米监控视频文件路径 |
|      uploadPath      |    上传路径    |
